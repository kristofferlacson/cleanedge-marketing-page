# CleanEdge Manpower Solutions - Marketing Landing Page

A professional, animated marketing landing page for CleanEdge Manpower Solutions.

## Features

- Responsive design
- Scroll-triggered animations
- Modern UI with professional styling
- Contact form
- Client showcase section
- Service cards with hover effects

## Technologies

- HTML5
- CSS3 (with animations)
- JavaScript (Intersection Observer API)
- Font Awesome icons
- Google Fonts

## Setup

1. Clone this repository
2. Open `index.html` in a web browser or serve via localhost
3. No build process required - pure static HTML/CSS/JS
